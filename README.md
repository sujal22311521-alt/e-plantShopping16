Paradise Nursery Shopping Application
📌 Project Description
The Paradise Nursery Shopping Application is a web-based platform that allows users to browse and purchase plants online. It provides an interactive interface for selecting plants, viewing details, and managing a shopping cart.

🌱 Features
Browse a variety of plants
View plant details and prices
Add items to cart
Manage shopping cart
User-friendly interface
💻 Technologies Used
HTML
CSS
JavaScript
